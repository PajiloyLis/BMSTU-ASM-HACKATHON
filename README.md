# Обзор проекта

Библиотека для работы с динамическими массивами в ОС DOS с модульными тестами. 

## Структура файлов

- `base_str.inc` - Включаемый файл структуры динамического массива
- `build.bat` - Batch-скрипт для сборки проекта
- `run.bat` - Batch-скрипт для запуска проекта
- `clear.bat` - Batch-скрипт для очистки артефактов сборки
- `alloc.asm` - Процедура выделения динамической памяти
- `clear.asm` - Очистка памяти и полей структуры
- `fmt.asm` - Форматированный вывод байтов динамического массива
- `get.asm` - Получение байта из динамического массива
- `set.asm` - Установка байта динамическому массиву
- `int.asm` - Обработчик прерывания int 21h функции ah=02h для тестирования
- `test_com.asm` - Функции для оформления тестирования
- `test_fmt.asm` - Тестирование функций форматированного вывода
- `test_get.asm` - Тестирование функций получения байта
- `test_set.asm` - Тестирование функций установки байта
- `unit.asm` - Модульные тесты

## Запуск проекта

Для запуска проекта выполните следующий шаг:

1. Запустите `run.bat` для запуска модульных тестов.

## Очистка проекта

Для очистки артефактов сборки выполните следующий шаг:

1. Запустите `clear.bat`, чтобы удалить временные файлы и подготовить проект к новой сборке.

## Сборка проекта

Для сборки проекта выполните следующие шаги:

1. Запустите `build.bat`, чтобы скомпилировать все файлы проекта и подготовить его к запуску модульных тестов.
